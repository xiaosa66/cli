### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `yCli` 命令了。

- `yCli init vue-template myVue`
- `yCli config get`
- `yCli config set type orgs`
- `yCli config set registry vuejs-templates`

- `yCli config set type users`
- `yCli config set registry YvetteLau`

### 发布

开发完成后，即可发布至 npm.
